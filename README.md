SVFGenerateImage 1.0

======================
簡単な図形をコードにより生成するユーティリティクラスを公開します。

 
使い方
------
SVFViewControllerを参照

2013-03-16現在
生成できる図形は、三角形、四角形、円形を基本図形として、三角形は（頂点が上下左右）と＞＜、四角形は角丸と角丸なし、です。

----------------
